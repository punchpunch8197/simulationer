# 시뮬레이셔너

안녕! 나는 이 게임을 만든 사람이야!

## 이게 뭐야?

이건 내가 만든 **캐릭터 만들기 게임**이야!
AI가 내가 설명한대로 캐릭터를 그려줘! 진짜 신기해!

## 할 수 있는 것들

### 1. 캐릭터 만들기
- 캐릭터 이름을 정하고
- 어떻게 생겼는지 설명하면
- AI가 그림을 그려줘!
- 마음에 들면 저장할 수 있어!

### 2. 맵 꾸미기
- 신비한 숲
- 미래 도시
- 우주 정거장
- 마법의 성
- 깊은 바다

이런 배경 중에서 고를 수 있어!
그리고 내가 만든 캐릭터를 맵에 올려놓을 수 있어!

## 어떻게 실행해?

```
npm install
npm run dev
```

이거 치면 돼!

## 만든 사람

나!

## 사용한 것들

- React (화면 만드는거)
- Vite (빠르게 만드는거)
- Tailwind CSS (예쁘게 만드는거)
- Supabase (저장하는거)
- Google Gemini (그림 그리는 AI)
